DISPLAY

La propiedad display define como es tratado un elemento y la disposicion de sus hijos

BLOCK

Se genera un salto de linea antes y despues del elemento, permite que se pueda usar las propiedades width y height

INLINE

El elemento no genera un salto de linea antes y despues, permitiendo poner varios elementos inline en una misma fila
